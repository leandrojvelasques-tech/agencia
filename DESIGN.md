# Design System for Leandro Velasques - Agencia IA

## Visual Language
- **Vibe:** Minimalist, technological, professional, and "human-centric" AI.
- **Style:** Clean lines, generous whitespace, subtle glassmorphism, and smooth transitions.
- **Theme:** Mostly Light/Clean with high-tech accents, but adaptable to a sleek "patagonia-dark" mode if needed.

## Color Palette
- **Primary Green (#285A47):** Used for primary branding, heavy headings, and trust elements.
- **Light Green (#A8D5C1):** Used for accents, soft backgrounds, and highlights.
- **Light Gray (#EDEDED):** Primary background color for a clean, professional look.
- **Dark Gray (#4F4C4D):** Text color and secondary background for contrast.

## Typography
- **Headings:** Modern, clean sans-serif (e.g., Inter or Outfit). Semi-bold for H1.
- **Body:** Highly readable sans-serif (e.g., Inter or Roboto).

## Components
- **Buttons:** 
  - Primary: Solid #285A47 with white text, rounded corners (8px). 
  - Secondary: Outlined #285A47 or Light Green #A8D5C1 background.
- **Cards:** White background with subtle borders or soft shadows.
- **Logo:** The Triskel logo should be used consistently as a symbol of growth and connection.
